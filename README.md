# LimaFlow - Sistema de Inteligencia y Analítica Vial

Proyecto para la gestión y análisis de datos de congestión vehicular en la ciudad de Lima.

## Enunciado del Proyecto

El sistema tiene como objetivo principal proporcionar a los gestores urbanos una herramienta analítica para detectar, visualizar y gestionar patrones de congestión, utilizando datos históricos y en tiempo real.

## Funcionalidades Core

1. **Sistema de Ingesta:** Captura de datos de congestión por tramos viales.
2. **Dashboard Analítico:** Visualización de zonas de congestión (mapas de calor) y tendencias temporales.
3. **Motor de Alertas:** Detección automática de anomalías y cuellos de botella.
